Log in

The program starts with a Log in screen, it will ask for an user and a password; once filled in, pressing Log in will check the credentials, the exit button closes the program.

If it is correct, the window on the left will appear, if it is incorrect, the window on the right will appear.

The following is the menu, depending on the privileges of the user, we will see different options, the admin will have user management and standard users will not; the logs button shows a log screen.

FTP Server

It will take us to the ftp window. Depending on the type of user, different buttons will appear.

File Management

Uploap File

You will see a drop-down menu in which you can choose the directory you want to upload it to, and the select file button will pop up a window that will allow you to choose the file to upload.

Download Files

A drop-down menu will appear to select the file from the server. When you click on the button, it will be downloaded.

Delete Files

The function is the same as above, but instead of downloading, it deletes the selected file.

Rename files

You have a dropdown that selects the file from the server, we put the name of the file and when you and when you click on the button you rename the file.

Directory Management

Create Directory

You will get a drop-down where you select the location and a text box for the name of the folder to create.

Delete Directory

Similar to above, but you select the folder to delete from a drop-down menu.

Acces Management

Select the folder from the drop-down menu. When you click on the button it tells you if you have permissions.

Storage (ADMIN)

In the drop-down menu you see the folders, in the text box you select the space in MB and the button modifies it. And at the bottom you get a table with the used space and its limit.

Permission(ADMIN)

In one dropdown you put the user/group and in the other the folder you want to give permissions to.
